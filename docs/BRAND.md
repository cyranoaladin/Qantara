# Marque Qantara AI

La charte de référence est dans `docs/qantara_ai_brand_kit`.

## Assets Versionnés

Les assets utilisés par l'application sont dans `public/brand` :

- `qantara-ai-logo-primary.svg` : header desktop sur fond sombre ;
- `qantara-ai-logo-light.svg` : usage ponctuel sur fond clair ;
- `qantara-ai-logo-monochrome.svg` : footer et contextes sobres ;
- `qantara-ai-mark.svg` : symbole compact mobile ;
- `qantara-ai-favicon.svg` : favicon officiel.

## Règles

- Ne pas déformer les proportions.
- Ne pas modifier les couleurs hors charte.
- Ne pas remplacer le logo par un texte brut dans le header.
- Ne pas ajouter de faux logos clients.
- Éviter les clichés IA : robots humanoïdes, cerveau lumineux, néons agressifs.

## Palette

- Background : `#050914`
- Background soft : `#08111F`
- Card : `#0B1220`
- Card alt : `#111827`
- Foreground : `#F8FAFC`
- Muted : `#94A3B8`
- Primary : `#22D3EE`
- Primary dark : `#0891B2`
- Secondary : `#D6A85A`
- Secondary dark : `#B9852F`

Les tests `brand-assets` vérifient que les chemins et tokens principaux restent
cohérents.
