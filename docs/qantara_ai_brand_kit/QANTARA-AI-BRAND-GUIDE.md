# Qantara AI — Charte graphique V1

## Concept

Le logo repose sur un signe simple : un **Q** géométrique contenant une **arche de pont**.

Le pont renvoie au nom Qantara et à la promesse de la société : faire le lien entre stratégie, formation, développement IA, automatisation et gouvernance.

## Positionnement visuel

- Sérieux
- Premium
- Scientifique
- Institutionnel
- Technologique sans excès
- Méditerranéen discret
- Adapté aux entreprises, écoles, institutions et clients francophones

## Typographies recommandées

- Interface : `Inter`, `Geist`, `IBM Plex Sans`, `Segoe UI`, `Arial`, sans-serif
- Titres : même famille, poids 700 à 800
- Corps : poids 400 à 500
- Métadonnées / badges : lettres légèrement espacées

## Usage du logo

### Versions fournies

- `qantara-ai-logo-primary.svg` : logo principal pour fond sombre.
- `qantara-ai-logo-light.svg` : logo pour fond clair.
- `qantara-ai-logo-monochrome.svg` : version blanche.
- `qantara-ai-mark.svg` : symbole seul.
- `qantara-ai-favicon.svg` : icône carrée pour favicon/app icon.

### Zone de protection

Laisser autour du logo un espace minimal équivalent à la hauteur du symbole intérieur du pont.

Ne pas coller le logo au bord d’une carte, d’un header ou d’une bannière.

### Tailles minimales

- Logo horizontal : largeur minimale recommandée 180 px en web.
- Symbole seul : 32 px minimum.
- Favicon : utiliser la version `qantara-ai-favicon.svg`.

### Interdictions

- Ne pas déformer le logo.
- Ne pas changer les proportions du symbole.
- Ne pas appliquer de rotation.
- Ne pas ajouter d’ombre forte.
- Ne pas placer la version colorée sur un fond très chargé.
- Ne pas remplacer le doré par un jaune vif.
- Ne pas remplacer le cyan par un bleu électrique agressif.

## Direction UI

### Fond

Utiliser un fond sombre dominant avec :

- halos cyan très discrets ;
- grilles fines ;
- bordures semi-transparentes ;
- cartes bleu nuit.

### Boutons

Bouton primaire :

- fond cyan `#22D3EE`
- texte navy `#050914`
- hover `#67E8F9`

Bouton secondaire :

- fond transparent
- bordure `rgba(148, 163, 184, 0.25)`
- texte `#F8FAFC`
- hover fond `rgba(148, 163, 184, 0.08)`

Bouton accent :

- doré `#D6A85A`
- usage limité : détails, badges, soulignements, pictogrammes.

## Ton visuel

Le site ne doit pas ressembler à :

- une agence marketing générique ;
- une startup IA flashy ;
- un site crypto ;
- une page cyberpunk ;
- un template SaaS impersonnel.

Il doit ressembler à :

- un cabinet IA rigoureux ;
- une structure de formation haut niveau ;
- un studio de développement fiable ;
- une entreprise capable d’adresser des dirigeants, enseignants et équipes techniques.
